# FFmpeg WinForms GUI 🎬

Простой, быстрый и легковесный графический интерфейс (GUI) для консольной утилиты FFmpeg, написанный на C# с использованием Windows Forms. 

Программа позволяет конвертировать видео- и аудиофайлы в популярные форматы в несколько кликов, избавляя от необходимости вручную писать сложные команды в терминале.

---

## ✨ Особенности и функции

* **Конвертация видео:** Перевод файлов в форматы **MP4**, **AVI**, **MKV**, **WebM**.
* **Экспорт в аудио:** Извлечение звуковых дорожек из видео и сохранение в **MP3**, **OGG**, **WAV**.
* **Оптимальные пресеты:** При конвертации в MP4 автоматически применяются кодеки `libx264` и `aac` с цветовым пространством `yuv420p` для гарантированного воспроизведения на стандартных плеерах Windows.
* **Анализ метаданных:** Программа автоматически считывает длительность медиафайла перед началом работы.
* **Фоновая обработка:** Процесс конвертации вынесен в фоновый поток (`BackgroundWorker`), благодаря чему интерфейс приложения не зависает во время работы.

## 🚀 Инструкция по запуску (Для пользователей)

Так как сам исполняемый файл `ffmpeg.exe` имеет большой размер и защищен авторскими правами, он не включен в этот репозиторий. Для работы программы выполните следующие шаги:

1. Перейдите в раздел **Releases** на GitHub и скачайте архив с готовой программой (или соберите её самостоятельно из исходного кода).
2. Скачайте официальную сборку FFmpeg для Windows (например, с сайта [ffmpeg.org](https://ffmpeg.org) или проверенного ресурса [gyan.dev](https://gyan.dev)).
3. Извлеките файлы `ffmpeg.exe`, `ffplay.exe`, `ffprobe.exe`, из скачанного архива.
4. **Важно:** Положите файл `ffmpeg.exe`, `ffplay.exe`, `ffprobe.exe`, в ту же самую папку, где находится исполняемый файл приложения (рядом с `.exe` файлом вашей программы).
5. Запустите программу, выберите файл, укажите желаемый формат и нажмите кнопку конвертации.

## 🛠 Как собрать проект (Для разработчиков)

Если вы хотите изменить интерфейс или доработать логику приложения:

1. Склонируйте данный репозиторий к себе на компьютер:
   ```bash
   git clone https://github.com
   ```
2. Откройте файл решения (Solution) в **Visual Studio** (рекомендуется версия 2013 и новее).
3. Убедитесь, что у вас установлена рабочая нагрузка для разработки настольных приложений `.NET Desktop Development`.
4. Нажмите **Сборка -> Собрать решение** (Build -> Build Solution).
5. Перейдите в папку `bin/Debug/` или `bin/Release/`, положите туда файл `ffmpeg.exe` и запустите приложение для тестирования.

## 💻 Технические детали реализации

Проект написан на классическом стеке Windows Forms и демонстрирует базовые принципы взаимодействия C# с внешними консольными утилитами:
* Взаимодействие с FFmpeg происходит через стандартный класс `System.Diagnostics.Process`.
* Длительность исходного медиафайла вычисляется путем перехвата и парсинга технического вывода `StandardError` при вызове команды `-i "путь_к_файлу"`.
* Длительные операции конвертации изолированы от UI-потока с помощью компонента `BackgroundWorker`, что обеспечивает отзывчивость интерфейса.
